# 生词本复习模式与词级朗读高亮设计

## 背景

当前 YomuTomo 已经具备以下能力：

- 阅读页支持句子播放、全文朗读和当前句子高亮
- 生词本已持久化到数据库，并支持掌握状态切换
- 生词本页面当前仍以网格列表为主，缺少复习流程

本次需求包含两部分：

1. 给生词本页面增加复习模式 / 翻牌模式
2. 把阅读页朗读高亮升级为词级同步高亮

## 目标

- 让用户可以在生词本里快速进入复习模式，按卡片逐条浏览词条
- 在复习模式中保留现有的“已掌握 / 取消掌握”状态切换能力
- 朗读播放时，从“句子级高亮”升级到“句子内词块级高亮”
- 不破坏现有的阅读页、评测页和生词持久化逻辑

## 非目标

- 不新增后端数据表
- 不新增复习进度持久化逻辑
- 不实现间隔重复、错题本、打卡统计等高级复习系统
- 不改动全文正文中的 ruby 渲染结构

## 设计原则

- 优先复用现有页面和接口
- 优先最小改动，避免引入新状态源
- 当浏览器能力不足时，允许优雅降级
- 词级高亮只影响朗读播放态，不和识别高亮互相覆盖

## 复习模式设计

### 入口

在 `templates/vocabulary.html` 的统计区增加一个复习入口按钮，例如“开始复习”。

点击后进入复习模式，不离开当前页面。

### 视图结构

复习模式使用单张卡片视图，卡片包含：

- 正面：单词 + 假名
- 背面：单词 + 假名 + 释义

复习模式只消费当前列表中的词条数据，不单独请求接口。

### 交互

- `翻牌`：正反面切换
- `上一张` / `下一张`：切换当前词条
- `已掌握` / `取消掌握`：沿用现有 `/vocabulary/toggle` 接口
- `退出复习`：返回原来的网格视图

### 状态管理

复习模式使用前端本地状态，不写入数据库：

- `reviewMode`: 是否正在复习
- `reviewIndex`: 当前卡片索引
- `reviewFlipped`: 当前卡片是否已翻面

页面刷新后默认回到普通列表视图。

### 空态与边界

- 如果当前筛选结果没有词条，复习入口保持不可用或提示无词可复习
- 如果用户在复习模式中把当前词条标记状态切走，仍保持当前卡片可见，列表状态在下一次刷新时同步
- 如果词条列表为空，复习视图显示空态文案，不报错

## 词级朗读高亮设计

### 目标范围

只升级阅读页中的句子播放面板，不改正文全文内容区的高亮算法。

### 切分策略

优先使用浏览器原生 `Intl.Segmenter('ja', { granularity: 'word' })` 对句子做词块切分。

如果浏览器不支持，回退为当前句子级高亮，不影响可用性。

### 播放行为

- 全文播放仍按句子顺序播放
- 单句播放仍按单句播放
- 当一个句子开始播放时，当前句子高亮
- 在句子内部按词块推进高亮状态

### 高亮层级

播放中同时存在两层状态：

- 当前句子：维持现有 `.sentence-item.is-active`
- 当前词块：在当前句子文本中追加 `.word-is-active` 或等价类名

### 实现边界

- 词级高亮只作用于句子列表，不改正文 ruby DOM
- 句子列表需要可重新渲染，且保持高亮状态可恢复
- 停止播放时清除词块高亮，只保留非播放态

## 前端实现边界

### 生词本页面

`templates/vocabulary.html` 负责：

- 提供入口按钮和复习容器
- 继续渲染列表卡片
- 输出给 JavaScript 使用的词条数据

前端脚本负责：

- 列表模式与复习模式切换
- 翻牌和前后切换
- 调用已有的生词状态切换接口

### 阅读页

`static/js/pages/reading.js` 负责：

- 保持现有句子播放流程
- 在句子播放基础上增加词块推进逻辑
- 在不支持 `Intl.Segmenter` 时优雅回退

## 数据流

### 生词本复习

1. 后端渲染 `vocab_rows`
2. 前端保存当前展示词条集合
3. 用户进入复习模式
4. 当前卡片只从前端数组读取数据
5. 用户切换掌握状态时调用 `/vocabulary/toggle`
6. 返回结果更新页面状态，不额外刷新整页

### 朗读词级高亮

1. 页面初始化时构建句子列表
2. 播放时按句子创建 `SpeechSynthesisUtterance`
3. 句子开始后计算词块序列
4. 播放过程中推进当前词块索引
5. 更新当前词块 DOM 状态
6. 播放结束或停止时清理词块状态

## 错误处理

- `speechSynthesis` 不可用时继续保留页面浏览能力，并提示不支持网页语音播放
- `Intl.Segmenter` 不可用时回退到句子级高亮
- `/vocabulary/toggle` 失败时显示错误提示，保持原状态不静默失败
- 本地状态读取失败时回退为默认状态，不阻断页面使用

## 测试策略

### 生词本复习

- 复习模式进入 / 退出状态切换
- 翻牌前后内容是否符合预期
- 上一张 / 下一张边界行为
- 掌握状态切换后按钮文案和样式是否更新

### 词级高亮

- 句子切分是否保持原有行为
- `Intl.Segmenter` 可用时是否生成词块序列
- 不可用时是否回退到句子级高亮
- 播放开始、切换、结束、停止时高亮状态是否正确清理

## 回滚方案

- 如果复习模式交互不稳定，可仅保留列表模式并移除复习入口
- 如果词级高亮兼容性不足，可保留句子级高亮作为最终降级方案
- 两个功能都只改前端展示层，不影响数据库结构和已保存数据
